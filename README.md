# Cluu Nodes

The package provides nodes to access the cluu plattform.

Nodes:
- CluuQuery
- CluuInsert
- CluuUpdate
- CluuDelete
- CluuInvoke
- CluuCallAssistant

## Prerequisites

You need access to a cluu plattform.
- Username
- Password
- Client ID
- Client Secret
